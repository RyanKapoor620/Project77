Open Console



Visual Stuido Code is an app where you create a website using three programming languages : CSS-Cascading Style sheets , HTML-Hyper text markup language and JS-Javascript.

A console takes input from the code you write and displays the output inside the console screen. To use and check the console , you have to right click the screen anywhere and click input from the options and you will find youself in the elements screen where it will show the creator's code of the website you are using . In the elements screen , click console on the top and you will be in the console screen.
